User Experience (UX) Design

Web Design £70

About this course

User experience (UX) design is fundamental to digital design, and an increasingly sought after skill to have amongst designers. This course provides an introduction to cutting edge technologies used in web design and development. It will teach you to apply the best practices in content layout and a key issue in web design is how the user feels, it is essential to enhance the experience of visitors to the website.

Skills required

No previous Experience of Web Design needed. You should have basic knowledge of computers and web browsers i.e. able to search on google and how to use keyboard and mouse.
